Lexita dokumentų valdymo sistema

1. Dokumentų įkėlimas naudojantis smart box ir automatizuotas duomenų nuskaitymas

Lengvai įkelkite dukumentus nusiųsdami juos į elektroninį paštą. Sistema pati nuskaitys pagrindinius duomenis ir jum jų nebereikės suvedinėti.

2. Nuosekli darbo eiga

Nuosekliai patvirtinkite dokumentus. Matykite kas jų nepatvirtino.

3. Greitesnis priėjimas prie bet kokio dokumento

  • Filtravimas pagal dokumento tipą;
  • Filtravimas pagal statusą;
  • Paieška pagal datą, serijinį numerį, atsakingą asmenį ar pavadinimą.

4. Lankstus teisių priskirimas

Priskirkite teises pagal atskiras vartotojų grupes arba pagal konkrečius asmenis

Sistema veikia remiantis web technologijomis

Veikia ant Windows, Linux , Mac os operacinių sistemų

Veikia su Chrome, Microsoft Edge, Mozilla Firefox, Safari naršyklėm
